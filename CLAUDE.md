## ディレクトリ構成

このリポジトリはフロントエンドとバックエンドをモノレポで管理する。

```text
/
├── frontend/   # React Native / Expo アプリ (TypeScript)
├── backend/    # Python / FastAPI サーバー
├── docs/       # 仕様書・アーキテクチャドキュメント
└── design/     # デザインファイル・スタイルガイド
```
